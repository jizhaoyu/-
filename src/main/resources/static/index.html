05 / Playbook

联调提示

把它当作联调脚本：先环境、再导入、再索引、最后问答验证。

  • 先看“运行态检查”卡片，如果 MySQL、Milvus 或 AI Provider 不是可用状态，优先处理环境问题。
  • 上传后若状态长期停留在 PROCESSING，先检查数据库、向量库和 AI 配置。
  • 问答接口没有命中内容时，会返回降级提示，而不是空页面。
  • 页面根路径 / 就是本地 demo 控制台，列表和问答默认共用同一个知识库 ID。
